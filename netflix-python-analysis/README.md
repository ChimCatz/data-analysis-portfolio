# netflix-python-analysis

This folder contains the first analysis in this repository: exploration and visualizations of Netflix Movies & TV Shows data (2010–2025).

Included files

- `netflix_movies_detailed_up_to_2025.csv` — movies dataset
- `netflix_tv_shows_detailed_up_to_2025.csv` — TV shows dataset
- `netflix_analysis.py` — main analysis runner/script
- `genre_popularity_vs_rating_scatter.py` — scatter visualization script
- `genre_share_treemap.py` — treemap visualization script
- `genre_top_genres_bar_chart.py` — top genres bar chart
- `analysis/` — notes and outputs for this analysis
- `visualizations/` — generated images (committed or generated locally)

How to run

1. Create and activate a virtual environment.
2. Install dependencies: `pip install -r requirements.txt` from the repository root.
3. Run scripts from this folder, or open them in an IDE to run cells.

Notes

- Data files are included for reproducibility. If you prefer not to commit large data, move them to `.gitignore` and download externally.
